## Why?

To learn deep aspects of css (or not), that's all.

## Features

- Themeable
- Responsive classes
- Pseudo classes

```html
<!-- responsive classes -->
<div class="flex md:flex-column xl:flex-row">...</div>

<!-- pseudo classes -->
<button class="interactive-01 hover:interactive-01-hover">...</button>
```
